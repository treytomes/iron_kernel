# Iron Kernel

Iron Kernel is an living retro computing platform focused on clear architecture, strong invariants, and long‑term maintainability.

This project explores what a kernel looks like when:
- design constraints are explicit,
- structure matters more than features,
- and evolution is guided by invariants rather than convenience.

Iron Kernel is not production‑ready. It is a research and development effort.

---

## Project Status

**Status:** Early development / experimental  
**Audience:** Systems developers, researchers, and contributors interested in kernel architecture and design

Expect:
- breaking changes
- incomplete subsystems
- evolving documentation

## Non‑Goals

Iron Kernel is **not**:
- a Linux replacement
- a compatibility layer
- a fast path to production use
